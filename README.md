A simple rust app for tasks.

USAGE

taskr --add "Go shopping" "2021-06-11"
Added task!

taskr --add "Go gym" "2021-03-11"
Added task!

taskr --done "Go shopping"
Deleted task!

taskr --list

This will lists tasks with the most recent at the bottom of the output
