A simple rust app for tasks.

USAGE

taskr --add "Go shopping" "2021-06-11"
Added task!

taskr --add "Go gym" "2021-03-11"
Added task!

taskr --done "Go shopping"
Deleted task!

taskr --list
"Go gym" - "2021-03-11"


taskr --upcoming
❗3
📅 5


the ! is for tasks due today or tomorrow
the candendar is for tasks due within 7 days

taskr --clear
Deleted all tasks!
